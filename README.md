# Data-library
A storage of valuable datasets across various sectors(Finance, Logistics, Retail, and Real-estate), that have been structured, cleaned, and analysed by me.
## Sectors Covered
- Logistics
- Finance
- Retail
- Real Estate

## Geographies
- Nigeria (primary — NBS, CBN, EFInA, NPA)
- United Kingdom (ONS, HMRC, Land Registry, FCA)
- International (World Bank, IMF, UN datasets)

## Repository Structure
Organised by sector → geography → dataset.
Each dataset contains:
- /raw — original file, never modified
- /cleaned — cleaned, analysis-ready version
- README.md — source, cleaning steps, 
  key variables, and use cases
